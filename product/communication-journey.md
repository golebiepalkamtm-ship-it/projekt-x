# J-004: Consented Communication

Communication begins only when both policy and participant controls allow it. The recipient’s choices—allow, limit, mute, block, report—are discoverable at the point of contact. A block takes effect across delivery paths promptly and does not disclose the blocker’s action to the other party.

Messages have delivery states without leaking recipient presence. Abuse controls include rate limits, attachment scanning, link handling, report context, and operator escalation. Retention, export, and lawful access rules are defined before implementation.
