# J-002: Edit and Publish a Profile

The editor saves drafts deliberately, shows validation at the field and summary level, and labels each field’s visibility. It never changes a public projection until the participant confirms publication.

Before publishing, the participant sees a preview, required verification/consent status, policy warnings, and effects on discovery. The system records the submitted version. If review is required, existing public material stays stable unless safety policy requires immediate restriction.

Autosave failure is visible, recoverable, and does not pretend data is stored. Concurrent edits use revision tokens and offer a compare-and-resolve state.
