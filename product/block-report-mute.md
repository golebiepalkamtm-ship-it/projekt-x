# Block, Report, and Mute

| Action | Effect | Visibility to target |
|---|---|---|
| Mute | suppresses non-critical notifications | not disclosed |
| Block | prevents policy-defined interaction and discovery paths | not disclosed |
| Report | creates a confidential safety intake | not disclosed by default |

Reports capture category, optional narrative, consent to share relevant context, and immediate-safety guidance. The reporter receives a reference ID and can add information without seeing restricted case data.
