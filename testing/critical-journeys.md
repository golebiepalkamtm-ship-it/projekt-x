# Critical Journey Test Register

| Journey | Minimum automated checks |
|---|---|
| Account access | enumeration resistance, MFA/recovery, session revocation |
| Profile publication | visibility projection, validation, revision conflict |
| Discovery | eligibility filter before ranking, privacy-safe zero results |
| Communication | consent, block propagation, report context |
| Case management | reason/version/audit trail, appeal compensating action |
| Commerce | idempotent intent, webhook replay, balanced ledger |
| Account closure | revocation, export, deletion workflow tracking |
