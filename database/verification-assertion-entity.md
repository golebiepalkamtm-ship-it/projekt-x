# Entity: Verification Assertion

**Owner:** Identity. An assertion records that an approved verifier evaluated a bounded claim at a point in time. Attributes include subject, claim type, result, issuer, evidence reference, issued/expiry/revocation time, and review reference.

Consumers receive only a scoped assertion. Raw evidence is segregated, encrypted, access-logged, and removed or retained according to its approved purpose. Assertions are immutable; corrections create a superseding assertion.
