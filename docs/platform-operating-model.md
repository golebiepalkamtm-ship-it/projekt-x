# Platform Operating Model

The platform is operated by Super Admin, Moderation, Support, Finance, Marketing, Content, AI Review, Verification, Fraud, Security, Privacy, and SRE/Platform functions. Access is role- and attribute-based, purpose-bound, time-limited for sensitive actions, and fully audited.

Every module defines on-call ownership, escalation path, service objective, policy owner, data owner, and operational runbook. Cross-functional incidents use one incident commander and preserve evidence before remediation.
