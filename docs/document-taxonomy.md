# Document Taxonomy

| Location | Purpose | Approval |
|---|---|---|
| `adr/` | Irreversible or expensive technical/product decisions | architecture owner |
| `rfc/` | Proposals requiring cross-functional review | named sponsors |
| `architecture/` | System boundaries and runtime contracts | architecture |
| `product/` | User behavior and business rules | product |
| `database/` | Data ownership, invariants, retention | data architecture |
| `security/` | Threats, controls, incidents, privacy | security |
| `docs/` | Cross-cutting project operation | project owner |
